Original Composition -
新香港 (Shin Honkon ~ Neo Hong Kong)

Finished on: 3/6/2025

.mp3 file

My original composition using the exact instruments used in the Capcom Power System II arcade game STREET FIGHTER
ZERO 3. I gave this track a slight asian style and a future-electronic theme that would suit for a neon-lit, futuristic Hong Kong. At the same time, I thought of this as a hypothetical theme for the character Yun, debuting in STREET FIGHTER III: NEW GENERATION (a year before SFZ3) and was added in the later final version of SFZ3 named STREET FIGHTER ZERO 3↑↑ (Double Upper).
